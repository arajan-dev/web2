GRID PROPERTIES (CONTAINER)

display
defines the element as a grid container
.container {
display: grid | inline-grid;
}
gird-template-columns
property specifies the number (and the track size) of columns in a grid layout
.container {
display:grid;
grid-template-columns: repeat(12,1fr);
}
gird-template-rows
property specifies the number (and the track size) of rows in a grid layout
.container {
display:grid;
grid-template-rows: 1fr 4fr;
}
grid-template
shorthand property for grid-template-columns and grid-template-rows
.container {
display: grid;
grid-template: repeat(12,1fr) / auto auto auto;
}
grid-template-areas
property specifies areas within the grid layout
.item {
grid-area: myArea;
}
.container {
display: grid;
grid-template-areas: "myArea myArea .";
}
gird-auto-rows
property sets a size for the rows in a grid container
.container {
display: grid;
grid-auto-rows: 100px;
}
gird-auto-columns
property sets a size for the columns in a grid container
.container {
display: grid;
grid-auto-columns: 1fr;
}
gird-auto-flow
property controls how auto-placed items get inserted in the grid
.container {
display: grid;
grid-auto-flow: row;
}
gird-gap
property specifies the size of columns and row gutters
.container {
display:grid;
gird-gap: 5px;
}

GRID PROPERTIES (ITEMS)

gird-column-start
Determines a grid item's location within the grid by referring to the column where it starts
.item {
grid-column-start: 2;
}
gird-column-end
Determines a grid item's location within the grid by referring to the column where it ends
.item {
grid-column-end: 5;
}
gird-row-start
Determines a grid item's location within the grid by referring to the row where it starts
.item {
grid-row-start: 3;
}
gird-row-end
Determines a grid item's location within the grid by referring to the row where it ends
.item {
grid-row-start: 6;
}
gird-row
shorthand property for gird-row-start and grid-row-end
.item {
grid-row: 3 / 6;
}
gird-column
Determines a grid item's location within the grid by referring to the row where it ends
.item {
grid-column: 2 / 5;
}
gird-area
Gives an item a name so that it can be referenced by a template created with the grid-template-areas property
.item {
grid-area: 2 / 1 / span 2 / span 3;
}
